Preguntas Frecuentes

→ ¿Cómo empiezo con el widget?

Puedes seguir nuestra guía paso a paso en la pestaña Empieza aquí o ver el video tutorial. Una vez que estés listo, ve a la pestaña Setup para generar tu widget ingresando tu token de integración de Notion y la URL de la base de datos. Todo el proceso toma aproximadamente 3 minutos.

→ ¿Cómo configuro correctamente mi base de datos de Notion?

Tu base de datos debe incluir estas tres columnas con nombres y formato exactos:
→ Name – Text
→ Publish Date – Date
→ Attachment – Files & Media

Para usuarios Pro, también agrega:
→ Image Source – Select [Image Attachment / Link / Canva Design]
→ Link – Text
→ Canva Link – URL

Simplemente sigue el formato del template predefinido.

→ ¿Por qué sigue diciendo que el widget no pudo acceder a mi base de datos de Notion?

Generalmente el enlace de base de datos no es el correcto. Asegúrate de copiar la URL de la base de datos real, no la página que la contiene. Abre la base de datos en vista de página completa y copia el enlace desde el navegador. Verifica además que usas la base de datos fuente, no una vista vinculada. Si ves una flecha antes del título, es una vista enlazada. Vuelve a la base de datos original y copia ese enlace. Una vez tengas el enlace correcto, debería funcionar sin problemas.

→ ¿Puedo usar este widget con mi base de datos existente?

Sí. Puedes usar cualquier base de datos que ya tengas en Notion. Solo asegúrate de que los nombres de las columnas coincidan exactamente con el template predefinido. Mantén tus demás columnas como quieras. Importante: cada template puede ubicar la base de contenido en lugares distintos, así que localiza la base de datos fuente y usa ese enlace para configurar tu widget.

→ ¿Puedo renombrar las columnas de la base de datos?

No. Las columnas deben mantener sus nombres exactos. Requeridas: Name, Publish Date, Attachment. En Pro también: Image Source, Link y Canva Link. Puedes agregar columnas adicionales, pero no renombres las especiales usadas por el widget.

→ ¿Puedo usar este widget para múltiples clientes?

Sí. Crea widgets separados para cada cliente:
→ Cada widget se conecta a una base de datos específica.
→ Para múltiples clientes, crea una base de datos independiente por cliente.
→ Puedes reutilizar el mismo token de integración en todas las bases (conéctalo a cada una).
→ Cada cliente obtiene su propia URL de widget.

→ ¿Cómo incrusto diseños de Canva? (Solo Pro)

Selecciona Canva Design en Image Source. En tu diseño de Canva, ve a Compartir → Embed y copia el enlace de incrustación. Pégalo en Canva Link. Nota: no sirve la URL del navegador; debe ser el enlace de Embed.

→ ¿Cómo agrego videos a mi galería? (Solo Pro)

Carga archivos de video en Attachment o pega enlaces de video en Link. Elige el tipo correcto en Image Source y el widget mostrará el video automáticamente con indicadores de reproducción.

→ ¿Cómo elimino el widget?

En Notion, elimina el bloque de incrustación. Para borrarlo definitivamente de tu cuenta, ve a la pestaña Account y elimina el widget desde ahí. También puedes desconectar la integración en la configuración de tu base de datos en Notion.

→ ¿Puedo reutilizar mi token de integración de Notion para múltiples widgets?

Sí. Puedes reutilizar el mismo token. Solo asegúrate de conectar esa integración a cada base de datos que quieras usar con un widget.

→ ¿Puedo usar múltiples widgets con diferentes bases de datos?

Sí. Puedes crear múltiples widgets y conectar cada uno a una base de datos distinta.

→ ¿Cuál es la diferencia entre las versiones Basic y Pro?

Basic: Muestra imágenes solo desde Attachment (hasta 12 ítems).
Pro: Contenido ilimitado desde archivos en Attachment, enlaces externos y diseños de Canva incrustados. Además: ítems clicables con vista completa, carrusel para múltiples imágenes y opción de fijar posts en la parte superior.

→ ¿Cómo actualizo a Pro?

Compra la licencia Pro en la tienda de Shopify. Si usas el mismo correo de tu cuenta, tus widgets se actualizan automáticamente a las funciones Pro. No necesitas recrearlos: solo recarga la página.

→ ¿Ofrecen reembolsos?

Para mantenerlo simple y justo, no ofrezco reembolsos. La versión básica gratuita está disponible para que pruebes todo antes de actualizar a Pro. Si tienes algún problema, escríbeme y te ayudo.

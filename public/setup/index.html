Planifica tu feed de Instagram trayendo el Grid a Notion

En pocos pasos tendrás una galería estilo Instagram que se actualiza automáticamente.

Pasos

1 Configura tu base en Notion

Empieza con nuestro template o crea tu base manualmente.

Opción 1: Duplicar template (recomendado)

Duplica el template para obtener una base lista con ejemplos.

Duplicar template →

Opción 2: Crear tu base desde cero

Usa exactamente estas propiedades:

PropiedadTipoUso
NameTextTítulo de cada item
Publish DateDateOrden del grid (más nuevo primero)
AttachmentFiles & MediaImágenes o videos
Image SourceSelectAttachment / Link / Canva Design (Pro)
LinkTextURL externa (Pro)
Canva LinkURLEnlace de incrustación de Canva (Pro)

Las tres primeras son requeridas. Las demás habilitan funciones Pro.

2 Crea tu integración y conéctala
  1. Abre Notion → My integrations y crea una “Internal Integration”.
  2. Copia el Integration Token.
  3. En tu base duplicada: menú ••• → Connections → selecciona tu integración.
3 Genera el widget e incrústalo
  1. Ve a la pestaña Setup.
  2. Pega tu Integration Token y la Database URL (abrir DB en vista de página completa y copiar la URL real de la base).
  3. Haz clic en Generar Widget y luego pega esa URL en Notion con /embed.

Consejos: si ves errores de acceso, casi siempre es porque la URL apunta a una vista vinculada o a una página que contiene la base, no a la base fuente.

Widget Setup

+ Instrucciones
  1. Crear integración interna en Notion.
  2. Copiar el token y pegarlo aquí. Puedes reutilizar el mismo token en futuros widgets.
  1. Abrir la base en Notion → ••• → Connections → seleccionar tu integración.

Debes copiar la URL de la base fuente abierta en vista de página completa.

+ Mostrar instrucciones
  1. Abre la base de datos en su propia página.
  2. Usa “Copy link” en el menú ••• o copia desde la barra del navegador.
  3. Evita enlaces de vistas vinculadas o páginas contenedoras.

Preguntas Frecuentes

¿Cómo empiezo con el widget?

Sigue la guía en Empieza aquí y luego genera tu widget en Setup. Tardarás unos minutos.

¿Cómo configuro correctamente mi base de Notion?

Usa el template o crea las columnas: Name (Text), Publish Date (Date), Attachment (Files & Media). Pro: Image Source (Select), Link (Text), Canva Link (URL).

El widget no puede acceder a mi base

Casi siempre es por usar la URL de una vista o de una página contenedora. Abre la base fuente en página completa y copia esa URL. Verifica también que la integración esté conectada en Connections.

¿Puedo usar múltiples widgets o múltiples clientes?

Sí. Crea una base por cliente, conecta la misma integración a cada base y genera un widget por base.

¿Cómo actualizo a Pro?

Compra la licencia Pro en tu tienda de Shopify. Si compras con el mismo correo, tus widgets se actualizan al refrescar.
